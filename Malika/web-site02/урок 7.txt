Закругление краев

border-radius: px, % ; - задает скругливание краев объекта

Пример:
<style>
border-radius: 15px;
</style>

* Для "превращения" объекта из квадрата в круг необходимо скруглить края на половину,
для этого необходимо задать border-radius: 50%;

Односторонние отступы margin и padding

Для того чтобы задать отступ с одной конкретной стороны top, bottom, left, right (вверх, низ, лево, право)
необходимо добавить соответственную сторону через тире

Пример:

<style>
margin-top: 10px;     - задает внешний отступ сверху 10 px
margin-bottop: 15px;  - задает внешний отступ снизу 15 px

padding-left: 20px;  - задает внутренний отступ слева 20 px
padding-right: 5%;   - задает внутренний отступ справа 5%
</style>

* За один раз можно применить только одну сторону, тоесть чтобы сделать внешний отступ в 10px свеху и справа
margin-top-right: 10px; - работать НЕ будет, для создание такого отступа необходимо добавить 2 стиля:

margin-top: 10px;
margin-right: 10px;
Такой вариант будет работать

Минимальная допустимая высота

min-height - работает как и знакомый нам уже height - он задает высоту объекта, но если содержимое объекта
слишком большое и выходит за пределы заданной высоты, то объект автоматически принимает такую высоту, 
в которой поместится все содержимое объекта.

Пример:

Создадим 2 параграфа (синий и красный) с коротким содержанием, которое помещается в оба блока.
Оба блока будут иметь width: 300px; (ширину 300px)
Красному зададим height, а синему min-height в 30px.


<style>
	.red{
	    background-color: salmon;
	    width: 300px;
	    height: 30px;	    
	}
	
	.blue{
	    background-color: lightblue;
	    width: 300px;
	    min-height: 30px;
	}
</style>

<body>
	<p class="red">
           Короткий текст в красном блоке
        </p>
        
            
         <p class="blue">
           Короткий текст в синем блоке
         </p>
</body>
Сейчас разницы нет между height и min-height, и содержимое параграфов выглядит одинаковым.
Теперь увеличим содержание обеих параграфов.

<style>
	.red{
	    background-color: salmon;
	    width: 300px;
	    height: 30px;	    
	}
	
	.blue{
	    background-color: lightblue;
	    width: 300px;
	    min-height: 30px;
	}
</style>

<body>
	<p class="red">
           Привет, я длиный текст в красном блоке, и я выхожу за рамки своего блока. 
	   И это не очень красиво. 
        </p>
        
            
         <p class="blue">
          Привет, я длиный текст в синем блоке, и я НЕ выхожу за рамки своего блока. 
	  И это уже смотриться красивее. 
         </p>
</body>

Модульный блок DIV

<div>...</div>  -  модульный тег создает блок, который можно изменять вне зависимости от его содержания, 
без содержания и стиля этот блок будет "невидимым", для того чтобы увидеть его, необходимо указать его стиль

Пример:

<style>
	div{
	    width: 200px;
	    height: 200px;
	    background-color: red;
	    border-radius: 20px;
	}
</style>
<body>
	<div>
	</div>
</body>

Данный пример создаст красный квадрат размером 200х200px, со скругленными краями. 
!!! Обратите внимание, что тег div пустой, он ничего не содержит, но все равно будет отображаться на сайте !!!
